# Aplikasi-Laptop-Operasional-
Aplikasi Laptop Operasional adalah sistem manajemen aset laptop berbasis web yang dikembangkan menggunakan Google Apps Script dan Google Sheets sebagai database. Aplikasi ini membantu pengelolaan data laptop, pengguna, dan status pemakaian secara real-time, sehingga lebih efisien, terstruktur, dan mudah diakses tanpa server tambahan.
